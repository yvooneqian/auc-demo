🎯 AUC 之美 · 一眼看懂模型排序力

AUC = ROC 曲线下面积 = P( score(正) > score(负) )

随机抽一个正样本和一个负样本，模型给正样本打分更高的概率

📊 秒懂三段式

  • TPR（真正率）：抓到的正例 / 全部正例　→　越高越好
  • FPR（假正率）：误杀的负例 / 全部负例　→　越低越好
  • AUC：不同阈值下 (FPR, TPR) 连成的曲线下面积，综合评判排序能力
极端举例
AUC = 1　→　正负分数完全不重叠，完美排序
AUC = 0.5　→　正负分数混在一起，≈随机猜
AUC < 0.5　→　模型反向排序，把负例打分更高（可反转预测得到 AUC > 0.5）

🛠 代码 30 秒复现

from sklearn.metrics import roc_auc_score
auc = roc_auc_score(y_true, y_score)
print(auc)

📝 实例：体检异常指标识别

背景： 1000 人做血常规，其中 100 人真实患病（正例）。
模型输出： 每个人患病的概率分数。
结果：
  • AUC = 0.92　→　随机抽一个患者和正常人，92 % 情况下患者得分更高，模型优秀。
  • AUC = 0.51　→　几乎随机，需要重调特征或换算法。
